# WebReader - 网页阅读器

一个功能完整的网页阅读器，支持 EPUB 和 TXT 格式，采用 Vite + Vue3 + TypeScript 构建。

## ✨ 功能特性

### 核心功能
- 📚 **支持多格式**：支持 EPUB 和 TXT 文件上传和阅读
- 📖 **书架管理**：在线书架展示所有已上传的书籍
- 🔖 **目录导航**：完整的章节目录，支持快速跳转
- 💾 **进度记录**：自动保存阅读进度，下次打开自动恢复
- 📱 **响应式设计**：完美适配 PC、平板和手机等各种屏幕尺寸

### PC 端特性
- ⌨️ **快捷键翻页**：
  - 右方向键 / 空格：下一章
  - 左方向键 / 退格：上一章
- 🎯 **侧边栏目录**：随时查看和跳转章节
- 📊 **进度条控制**：直观的进度显示和拖拽调整
- 🖱️ **流畅交互**：优化的鼠标操作体验

### 移动端特性
- 👆 **触屏适配**：优化的触屏操作体验
- 🎨 **简洁界面**：隐藏式控制栏，点击显示
- 📜 **目录抽屉**：向上滑动展开目录列表
- 🔘 **按钮导航**：大按钮设计，易于点击

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

然后在浏览器中打开 `http://localhost:5173`

### 生产构建

```bash
npm run build
```

构建产物会输出到 `dist` 目录

### 预览生产构建

```bash
npm run preview
```

## 💾 数据存储

所有数据都存储在浏览器的 LocalStorage 中，包括：
- 书籍元数据（标题、文件名、添加时间等）
- 书籍内容（章节和文本）
- 阅读进度（当前章节、阅读位置）

**注意**：LocalStorage 有大小限制（通常为 5-10MB），建议不要添加过多或过大的文件。

## 📖 使用说明

### 上传书籍
1. 点击"📤 上传书籍"按钮
2. 选择 .epub 或 .txt 文件（支持多选）
3. 等待文件处理完成

### 阅读书籍
1. 在书架中点击想要阅读的书籍
2. 进入阅读器页面
3. 使用快捷键或按钮翻页

### 使用目录
- **PC**：点击右上角"☰ 目录"按钮，左侧会展开目录面板
- **移动端**：点击右上角"☰"按钮，会弹出目录抽屉

### 删除书籍
- 在书架中点击书籍卡片上的"删除"按钮
- 确认删除后，书籍会被移除

## 🔧 技术栈

- **框架**：Vue 3
- **构建工具**：Vite
- **语言**：TypeScript
- **EPUB 支持**：jszip
- **存储**：Browser LocalStorage

## 📝 文件格式支持

### EPUB
- 支持标准 EPUB 2.0 和 3.0 格式
- 自动提取章节信息
- 支持嵌入的图片和样式

### TXT
- 支持纯文本格式
- 自动识别常见的章节标记（第n章、Chapter n 等）
- 无法识别时按字数自动分割

## 🎨 响应式设计

| 设备类型 | 分辨率 | 布局 |
|---------|--------|------|
| PC | >= 1200px | 侧边栏 + 主内容 |
| 平板 | 768px - 1199px | 响应式网格 |
| 手机 | < 768px | 全屏阅读 + 抽屉菜单 |

## ⚠️ 注意事项

1. **浏览器支持**：推荐使用最新版本的现代浏览器
2. **存储限制**：LocalStorage 限制通常为 5-10MB，请合理控制上传文件大小
3. **隐私**：所有数据存储在本地浏览器，不会上传到服务器
4. **导出**：目前没有导出功能，数据仅在 LocalStorage 中
5. **格式**：请确保上传的文件格式正确，损坏的文件可能无法读取

## 🐛 已知问题

1. 大型 EPUB 文件可能需要较长的处理时间
2. 某些非标准 EPUB 文件可能无法正确解析
3. TXT 文件的章节识别可能不够准确
