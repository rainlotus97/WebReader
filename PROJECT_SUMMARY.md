## 🏗️ 技术架构

### 技术栈
- **前端框架**: Vue 3 (Composition API)

- **编程语言**: TypeScript 5
- **包管理**: pnpm
- **库**:

## 🔐 数据存储

### 存储结构
```javascript
// 书籍列表
localStorage['webreader_books'] = [
  {
    id: "string",              // 唯一 ID
    title: "string",           // 书名
    filename: "string",        // 原始文件名
    fileType: "epub|txt",      // 文件类型
    addedAt: number,           // 添加时间
    progress: {
      chapter: number,         // 当前章节
      position: number,        // 章节位置
      timestamp: number        // 更新时间
    }
  }
]

// 书籍内容
localStorage['webreader_${bookId}'] = [
  {
    id: "string",              // 章节 ID
    title: "string",           // 章节标题
    content: "string",         // 章节文本
    index: number              // 章节索引
  }
]
```

### 容量估算
- 元数据：通常 < 100KB
- 单本书内容：
  - 小说（50万字）：~2MB
  - 大型著作：~5-10MB
- 浏览器限制：通常 5-10MB

## 🐛 已知限制

### EPUB 解析
- 仅支持标准 EPUB 2.0/3.0
- 只提取文本，不保留图片/样式
- 复杂排版可能不完美

### TXT 解析
- 章节识别基于模式匹配
- 特殊格式可能识别不准
- 大文件分章可能不够精准

### 浏览器
- IE 11 不支持
- 移动浏览器差异（Safari、Chrome 表现可能略有不同）
