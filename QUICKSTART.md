# 快速开始指南

## 系统要求

- Node.js 18+ 或 pnpm 8+
- 现代浏览器（Chrome 90+、Firefox 88+、Safari 14+、Edge 90+）

## 安装步骤

### 1. 克隆或初始化项目

```bash
cd /Users/rainlotus/Desktop/web/WebReader
```

### 2. 安装依赖

```bash
pnpm install
```

### 3. 启动开发服务器

```bash
pnpm dev
```

浏览器会自动打开 `http://localhost:5173`

## 基本操作

### 1. 上传书籍

1. 点击页面右上角的 "📤 上传书籍" 按钮
2. 选择一个或多个 EPUB 或 TXT 文件
3. 等待文件处理完成
4. 书籍会自动添加到书架

### 2. 开始阅读

1. 从书架中点击任何书籍卡片
2. 书籍将在阅读器中打开
3. 享受阅读体验

### 3. PC 版本控制

#### 目录导航
- 点击左侧边栏中的章节进行跳转
- 或点击右上角的 "☰ 目录" 按钮

#### 快捷键翻页
- **下一章**：按 `→` 方向键或 `Space` 空格键
- **上一章**：按 `←` 方向键或 `Backspace` 退格键

#### 鼠标操作
- 拖动底部进度条快速跳转
- 点击 "上一章" / "下一章" 按钮翻页

### 4. 移动版本控制

#### 显示控制栏
- 点击阅读区域任意位置显示/隐藏底部控制栏

#### 打开目录
- 点击右上角 "☰" 按钮打开目录抽屉
- 从列表中点击章节进行跳转

#### 翻页
- 使用底部的 "← 上一章" 和 "下一章 →" 按钮

### 5. 管理书籍

#### 返回书架
- 点击阅读界面左上角的 "← 返回书架" 按钮

#### 删除书籍
- 在书架页面，点击书籍卡片上的 "删除" 按钮
- 确认删除（不可恢复）

## 进度自动保存

✅ 所有阅读进度都会自动保存到浏览器本地存储

- 翻页时自动保存
- 关闭浏览器后数据不丢失
- 下次打开时自动恢复到上次阅读位置

## 支持的文件格式

### EPUB 电子书
- 标准 EPUB 2.0/3.0 格式
- 自动识别章节结构
- 提取和清理文本内容

### 纯文本 TXT
- UTF-8 编码的文本文件
- 自动识别章节标记：
  - `第[数字]章`、`第[中文数字]章`
  - `Chapter n`、`chapter n`
  - `[数字].` 格式
- 如果识别不到章节标记，将按 5000 字自动分章

## 常见问题

### Q: 上传的书籍保存在哪里？

A: 书籍数据存储在浏览器的 localStorage 中。
- 位置：浏览器开发者工具 → Application → Local Storage
- 键名：以 `webreader_` 开头
- 容量限制：通常 5-10MB

### Q: 刷新页面后书籍会丢失吗？

A: **不会**。所有书籍数据都持久化存储在本地，刷新后可以恢复。

### Q: 如何彻底删除所有书籍？

A: 
**方法 1：逐一删除**
- 在书架中点击每本书的 "删除" 按钮

**方法 2：清除浏览器数据**
- 打开浏览器开发者工具（F12）
- Application → Local Storage
- 删除域名下所有 `webreader_` 开头的项

### Q: 大文件会不会卡顿？

A: 
- EPUB 文件通常 1-10MB，能正常处理
- TXT 文件几乎无大小限制
- 超大文件（>50MB）可能会较慢

### Q: 支持离线阅读吗？

A: 当前版本需要网络。书籍内容保存在本地，但应用代码需要从服务器加载。

### Q: 能导出阅读进度吗？

A: 不能直接导出，但可以：
- 手动记录 localStorage 中的数据
- 使用浏览器导出功能
- 提交 Issue 请求该功能

## 高级技巧

### 1. 批量上传

同时选择多个文件进行上传：
- 点击 "📤 上传书籍"
- 按住 Ctrl（Windows）或 Cmd（Mac）
- 点击多个文件
- 确认上传

### 2. 快速访问

将网站添加到书签栏，快速访问阅读器。

### 3. 全屏阅读

使用浏览器的全屏功能（F11）进行沉浸式阅读。

### 4. 调整字体大小

使用浏览器缩放功能：
- `Ctrl/Cmd + +` 放大
- `Ctrl/Cmd + -` 缩小
- `Ctrl/Cmd + 0` 重置
